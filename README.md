# AI Foundation Series - Code Repository

This repository contains the code examples for the AI Foundation Series blog posts. Each series installment has its own folder with complete, working code that you can run and learn from.

## Repository Structure

```
foundation_series/
├── README.md                 # This file
├── 1-foundation/            # Series 1: Basic AI chatbot
├── 2-foundation/            # Series 2: (Coming soon)
├── 3-foundation/            # Series 3: (Coming soon)
└── ...
```

Each folder contains the complete code for that part of the series, along with its own README with specific setup instructions.

## Getting Started

### Prerequisites

- [Python 3.13+](https://www.python.org/downloads/)
- [uv](https://docs.astral.sh/uv/getting-started/installation/) package manager
- Basic terminal/command line skills

### Optional (depending on LLM provider)

- [Ollama](https://ollama.com/download) for local models
- OpenAI API key
- Anthropic API key

### Quick Start

1. Clone this repository:

   ```bash
   git clone https://github.com/maglore9900/foundation_series.git
   cd foundation_series
   ```
2. Navigate to the series you want to work with:

   ```bash
   cd 1-foundation
   ```
3. Follow the README in that folder for setup and usage.
